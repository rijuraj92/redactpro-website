FAQ
Questions from Legal & Security Teams
Is my data safe?
RedactPro runs 100% offline on your local machine. No documents are uploaded anywhere. No telemetry, no analytics, no cloud processing. Your files never leave your network.
Are the redactions truly permanent?
Yes. Redacted text is physically removed from the PDF using PyMuPDF black-box overlays. The original text is not hidden — it's deleted. Additionally, all metadata is stripped from output files.
How does AI PII detection work?
RedactPro uses Microsoft Presidio, an open-source NLP engine, running locally on your machine. It identifies PII using statistical models and pattern recognition — no data is sent to any API or cloud service.
Can I use it with Purview eDiscovery exports?
Yes. Export your search results from Microsoft Purview eDiscovery, then import the entire export folder into RedactPro. It handles MSG, EML, and all common formats including renamed file types.
What about scanned PDFs?
RedactPro includes Tesseract OCR. Scanned pages are rendered at 300 DPI, OCR'd locally, and the extracted text is available for redaction. No cloud OCR services are used.
Are redaction certificates legally defensible?
Each export generates a certificate with a unique ID, timestamp, operator identity, SHA-256 hashes of original and redacted files, and a complete log of all redaction actions. This provides a verifiable chain of custody.
Does it handle emails with image-only bodies?
Yes. RedactPro detects emails with image attachments instead of text bodies. It lists attachments in the output and applies OCR to extract and redact text from embedded images when possible.
Can multiple team members use it?
Each license is per-user. For teams of 5+, contact us for volume pricing with shared keyword profiles and priority support. Annual plans include invoice billing for procurement.
